## Getting Started

### Logging In
1. Navigate to `http://your-domain.com/login`
2. Enter your email and password
3. Click "Log in"
4. You will be redirected to the admin dashboard

### Default Credentials
- Email: `admin@pjs-law.com`
- Password: `password`

**⚠️ Change your password immediately after first login!**

## Dashboard Overview

The dashboard displays:
- **Total News Articles** - Count of all news articles
- **Total Documents** - Count of all downloadable documents
- **Total Case Studies** - Count of all case studies
- **Total Team Members** - Count of team members
- **Unread Contacts** - Number of unread contact messages

Quick action buttons provide direct access to create new content.

## Managing News

### Creating News Article
1. Click **"News"** in the navigation menu
2. Click **"Create New"** button
3. Fill in the form:
   - **Title (Thai)** - Required
   - **Title (English)** - Required
   - **Slug** - Auto-generated from English title (can be edited)
   - **Excerpt (Thai)** - Short summary in Thai
   - **Excerpt (English)** - Short summary in English
   - **Content (Thai)** - Full article content in Thai (Required)
   - **Content (English)** - Full article content in English (Required)
   - **Featured Image** - Upload an image (JPG, PNG)
   - **Category** - Select from dropdown
   - **Published** - Check to make article visible on website
   - **Published Date** - Set publication date
4. Click **"Create News"** button

### Editing News Article
1. Go to **News** page
2. Click **"Edit"** button on the article you want to edit
3. Modify the fields
4. Click **"Update News"** button

### Deleting News Article
1. Go to **News** page
2. Click **"Delete"** button on the article
3. Confirm deletion in the popup
4. Article will be permanently deleted

### Searching and Filtering
- Use the search box to find articles by title
- Filter by category using the dropdown
- Filter by published status (All, Published, Draft)

## Managing Documents

### Uploading Documents
1. Click **"Documents"** in the navigation
2. Click **"Create New"** button
3. Fill in the form:
   - **Title (Thai)** - Required
   - **Title (English)** - Required
   - **Description (Thai)** - Brief description
   - **Description (English)** - Brief description
   - **File** - Upload PDF, DOC, or XLS file (Required)
   - **Category** - Select category
   - **Active** - Check to make document downloadable
4. Click **"Upload Document"** button

### Editing Documents
1. Go to **Documents** page
2. Click **"Edit"** on the document
3. Update information
4. Optionally upload a new file (this will replace the old file)
5. Click **"Update Document"**

### Deleting Documents
1. Go to **Documents** page
2. Click **"Delete"** on the document
3. Confirm deletion
4. Document file will be permanently removed

## Managing Case Studies

### Creating Case Study
1. Click **"Case Studies"** in navigation
2. Click **"Create New"** button
3. Fill in the form:
   - **Title (Thai/English)** - Required
   - **Slug** - Auto-generated
   - **Client Name** - Can use "Confidential" if needed
   - **Challenge (Thai/English)** - Describe the problem
   - **Solution (Thai/English)** - Describe how you solved it
   - **Result (Thai/English)** - Describe the outcome
   - **Featured Image** - Upload main image
   - **Category** - Select category
   - **Published** - Check to display on website
4. Click **"Create Case Study"**

### Editing Case Studies
Similar to news editing - click "Edit", modify fields, and save.

### Deleting Case Studies
Click "Delete" button and confirm.

## Managing Team Members

### Adding Team Member
1. Click **"Team Members"** in navigation
2. Click **"Create New"** button
3. Fill in the form:
   - **Name (Thai)** - Required (e.g., นายธนากร ตั้งกิจโสภา)
   - **Name (English)** - Required (e.g., Thanagon Tagkidsopha)
   - **Position (Thai)** - Required (e.g., ประธานบริษัท)
   - **Position (English)** - Required (e.g., President)
   - **Bio (Thai)** - Short biography
   - **Bio (English)** - Short biography
   - **Photo** - Upload profile photo
   - **Order** - Number for display order (1 = first)
   - **Active** - Check to display on website
4. Click **"Create Team Member"**

### Display Order
- Team members are displayed in ascending order
- Set Order = 1 for the most important person (will appear first)
- Set Order = 2, 3, 4, etc. for others

## Viewing Contact Messages

### Accessing Messages
1. Click **"Contacts"** in navigation
2. You'll see a list of all contact form submissions
3. Unread messages are highlighted

### Reading Messages
1. Click on any message to view details
2. Message will be automatically marked as read
3. You can see:
   - Sender's name
   - Phone number
   - Email (if provided)
   - Subject
   - Full message details
   - Submission date

### Deleting Messages
1. Open the message or from the list
2. Click **"Delete"** button
3. Confirm deletion

## Tips and Best Practices

### Content Tips
1. **Always fill both Thai and English fields** - Website supports language switching
2. **Use descriptive slugs** - Helps with SEO
3. **Optimize images** - Resize images before uploading (recommended: 1200x800px for featured images)
4. **Write clear excerpts** - These appear in listing pages
5. **Use categories consistently** - Helps users find content

### Image Guidelines
- **News Images**: 1200x800px, JPG or PNG
- **Team Photos**: 400x400px (square), professional headshots
- **Case Study Images**: 1200x800px minimum
- **File Size**: Keep under 2MB for faster loading

### Document Guidelines
- **File Types**: PDF preferred, DOC/XLS acceptable
- **File Names**: Use descriptive names
- **File Size**: Keep under 10MB when possible
- **Descriptions**: Write clear descriptions so users know what they're downloading

### SEO Best Practices
1. Use descriptive titles
2. Write compelling excerpts
3. Choose relevant categories
4. Set appropriate publish dates
5. Use keywords naturally in content

### Security
1. **Change default password** immediately
2. **Use strong passwords** (mix of letters, numbers, symbols)
3. **Don't share admin credentials**
4. **Log out when finished**
5. **Regularly update content** to keep website fresh

### Getting Help
If you encounter any issues:
- Check that all required fields are filled
- Ensure uploaded files meet size/type requirements
- Clear your browser cache if changes don't appear
- Contact technical support: pjs.legal2025@gmail.com
